## doutu-flow
support doutu for alfred.

## Usage
1. Keyword trigger
    > select a keyword content, then call alfred:`dt` to trigger doutu flow.
2. Hotkey trigger
    > select a keyword content, then press hotkey:`command+,` to trigger doutu flow.
